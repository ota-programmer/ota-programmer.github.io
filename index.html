For STM32 Microcontroller

This project is implemented to program BlueBill Board (STM32 Micro-controller) over the air using a website connected to a cloud database (Firebase) and NodeMCU board to get the code from internet for sending it to STM to write it to the Flash Memory using the Bootloader flashed on the first 8 kB of Flash Memory.

USAGE

Usage discreption.

  • The testing section sends to the NodeMCU and waits 5 seconds to respond to know if it is connected to Internet or not.

  • The Linker script file mem.id should be edited to change the flash size to 56KB as Boot Loader uses 8 KB and start address should be edited:

    FLASH (rx) : ORIGIN = 0x08002000, LENGTH = 56K

  • Uploading the code doesn't mean it's flashed, once the NodeMCU is connected to Internet and connected to STM it will send the uploaded code to STM to flash

  • The full code is on GitHub Organization (The link below)
